Silverback Documentation

Dive deep into the Silverback SDK and learn how to build powerful blockchain bots.

Installation

Get started with Silverback by following these steps:

  • Install Silverback: pip install silverback
  • Verify installation: silverback --version

Creating a Bot

Learn how to create and deploy a bot with Silverback:

  • Create a bot file: bots/MyBot.py
  • Deploy your bot: silverback run MyBot

Event Handling

Set up event handling for your bot:

  • Define event handlers in your bot file
  • Test your event handling: silverback test MyBot

Best Practices

Follow these best practices for building robust bots:

  • Use environment variables for sensitive data
  • Implement error handling and logging
