Getting Started

Your journey to Web3 development starts here.

Step 1: Set Up Your Environment

Create a Virtual Environment (Recommended)

# Create virtual environment
python3 -m venv /path/to/new/environment
source /bin/activate

Install Ape Framework

# Update pip and install Ape
pip install -U pip
pip install eth-ape

Step 2: Create Your First Project

Initialize a New Project

# Create a new project
ape init

Run Your First Script

# Deploy your first contract
ape run deploy

Installation

Get started with Ape Framework by following these steps:

  • Install Ape: pip install eth-ape
  • Verify installation: ape --version

Project Setup

Initialize and set up your Ape project:

  • Create a new project: ape init
  • Configure your project settings in ape-config.yaml

Writing Smart Contracts

Learn how to write and deploy smart contracts with Ape:

  • Create a contract file: contracts/MyContract.vy
  • Deploy your contract: ape run deploy

Testing

  • Write tests in tests/
